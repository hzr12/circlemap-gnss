# 同心圆地图雷达 — Circlemap

> 基于腾讯地图的同心圆雷达工具，专为"鬼抓人"等户外活动设计。  
> 以自己或任意点为中心，绘制多层同心圆，实时查看与圆心的距离、方位趋势。

## 功能

### 地图操作
- **点击选点** — 点击地图任意位置设为圆心
- **坐标输入** — 手动输入经纬度，支持智能粘贴解析
- **智能坐标解析** — 自动识别 `23.1291, 113.2644` / `39.9°N 116.4°E` / `lat 23.1291 lng 113.2644` / `N 39.9 E 116.4` 等多种格式
- **长按地图** — GPS 过期时设为手动位置，否则快速创建圆
- **URL 参数** — `?lat=39.9&lng=116.4&radius=1000` 直接打开指定位置

### GPS 定位
- **短按** GPS 按钮 → 单次定位，飞到当前位置
- **长按** GPS 按钮 → 切换持续追踪（后台自动暂停/恢复）
- **WGS84 → GCJ-02** 纯 JS 纠偏（中国坐标偏移算法）
- **精度环** 显示定位精度范围
- **定位过期提醒** — 10 分钟未更新自动提示
- **自动重定位** — 位置过期且未追踪时自动尝试

### 同心圆
- 对数半径滑块 — 常用小半径占据更多滑块行程
- 多层同心圆绘制（默认每 2.5km 一圈）
- 圆列表 — 查看、选中、编辑半径、删除
- 信息面板 — 圆心坐标、半径、面积、圈数

### 位置追踪
- **距圆心距离** — 实时显示每个圆的距离
- **范围内检测** — 是否进入圆圈范围
- **靠近/远离趋势** — 箭头指示运动方向
- **地图跟随模式** — 点击状态条切换，位置更新自动移动视角

### 轨迹记录
- 记录移动轨迹，自动采样（最小间隔 10m）
- 导出 **GPX** 文件（兼容主流导航应用）
- 轨迹距离统计

### 持久化
- 圆圈、半径、中心点自动保存到 localStorage
- 刷新页面自动恢复

### 最近定位
- 记录最近 10 次定位（含精度、时间）
- 手动定位标记 📍 区分

### 界面
- 深色主题 · 玻璃质感 · 移动优先
- 移动端面板折叠 — 点击把手收起/展开
- 响应横竖屏旋转

## 快速开始

1. 打开 `index.html`
2. 点击地图选点，或粘贴坐标
3. 调整半径，点击「绘制圆形」
4. 短按 GPS 按钮定位，长按开启持续追踪

## 技术栈

- 纯 JavaScript（ES6+，无框架）
- 腾讯地图 JavaScript API v2
- Canvas 叠加层绘制同心圆
- WGS84 ↔ GCJ-02 纯前端纠偏算法
- localStorage 持久化
- 深色主题 CSS 变量

## 浏览器支持

- Chrome / Edge / Safari（含 iOS）
- 现代移动端浏览器
